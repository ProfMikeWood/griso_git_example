# A Git/Hub Primer
The is an example repository to show some Git/Hub tools during the GRISO workshop.

## What is a GitHub repository used for?
GitHub repositories can be used for a variety of purposes! Here are a few examples of public repositories:
- Example of modeling source code: [MITgcm](https://github.com/MITgcm/MITgcm)
- Example of a software package: [diagnostics_vec](https://github.com/mhwood/diagnostics_vec) (by Mike Wood)
- Example of a project to support a publication: [sermilik_gridded_hydrography](https://github.com/a1roth/sermilik_gridded_hydrography) (by Aurora Roth)
- Example of notebooks to generate a "book": [ocean_modeling_book](https://github.com/ProfMikeWood/ocean_modeling_book) (by Mike Wood, see book [HERE](https://profmikewood.github.io/ocean_modeling_book/intro.html))
